                           1. ОБЩИЕ СВЕДЕНИЯ

1.1. НАЗВАНИЕ ПРОЕКТА
---------------------
Рабочее название: "RentFlow CRM" (или другое на выбор заказчика)
Альтернативы: AutoRent Pro, FleetMaster, RentEase

1.2. НАЗНАЧЕНИЕ САЙТА
---------------------
Создание презентационного сайта-лендинга для коммерческой CRM-системы
управления арендой транспортных средств. Сайт должен выполнять функции:
- Презентация продукта и его возможностей
- Привлечение новых клиентов (компаний-арендодателей)
- Регистрация и вход в CRM-систему
- Выбор тарифного плана
- Демонстрация возможностей (демо-доступ)
- Поддержка существующих клиентов

1.3. ЦЕЛЕВАЯ АУДИТОРИЯ
----------------------
- Компании по прокату автомобилей (от малого до среднего бизнеса)
- Каршеринговые сервисы
- Транспортные компании с арендой техники
- Прокат спецтехники, мотоциклов, велосипедов, скутеров
- Яхт-клубы и прокат водного транспорта
- Частные предприниматели в сфере аренды

1.4. ТЕХНОЛОГИЧЕСКИЙ СТЕК
-------------------------
Фронтенд:
- React 18+ с TypeScript
- Tailwind CSS (консистентность с основной CRM)
- Vite для сборки
- React Router DOM для маршрутизации
- Framer Motion для анимаций
- React Hook Form + Zod для форм

Бэкенд (интеграция с существующим):
- Express.js / Node.js
- PostgreSQL
- Расширение существующего API

Дополнительно:
- Интеграция платежных систем (Stripe, PayPal, ЮKassa)
- Email-сервис (SendGrid, Resend)
- Analytics (Google Analytics, Яндекс.Метрика)
