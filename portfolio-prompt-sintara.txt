# ПРОМПТ ДЛЯ ДОБАВЛЕНИЯ ПРОЕКТА UNICAR В ПОРТФОЛИО SINTARA

## Задача
Добавить проект UNICAR в раздел "Работы" на сайте IT-студии Sintara (sintara.io).

---

## ИНФОРМАЦИЯ О ПРОЕКТЕ

### Название проекта
**UNICAR** — Система управления арендой транспорта

### Тип проекта
Веб-приложение / Админ-панель / CRM-система для бизнеса

### Клиент / Локация
Компания по аренде автомобилей, Таиланд

### Краткое описание (для карточки)
Полнофункциональная CRM-система для управления автопарком и арендами. Включает dashboard с аналитикой, управление клиентами, автомобилями, арендами, техобслуживанием и финансовой отчётностью.

### Расширенное описание
UNICAR — это комплексная система управления бизнесом по аренде автомобилей, разработанная с нуля. Приложение позволяет полностью автоматизировать процессы: от оформления аренды до отслеживания техобслуживания и формирования финансовых отчётов. Интерфейс адаптирован для работы на любых устройствах и поддерживает тёмную тему.

---

## ВЫПОЛНЕННЫЕ РАБОТЫ

### Функциональные модули:
1. **Dashboard** — интерактивная главная панель со статистикой в реальном времени:
   - Активные аренды
   - Свободные автомобили
   - Доход за месяц
   - Ближайший возврат
   - Рейтинг популярности автомобилей
   - Система уведомлений

2. **Управление арендами** — полный цикл работы с арендами:
   - Создание и редактирование аренд
   - Отслеживание статусов
   - История аренд по клиентам и автомобилям

3. **Каталог автомобилей** — управление автопарком:
   - Карточки автомобилей с фото
   - Статусы доступности
   - История использования

4. **База клиентов** — CRM-функциональность:
   - Карточки клиентов с историей аренд
   - Контактная информация
   - Поиск и фильтрация

5. **Техобслуживание** — планирование и учёт ТО:
   - График обслуживания
   - История ремонтов
   - Напоминания

6. **Финансы и отчёты** — аналитика бизнеса:
   - Доходы и расходы
   - Отчёты по периодам
   - Статистика по автомобилям

7. **Настройки** — конфигурация системы

### UX/UI особенности:
- Современный минималистичный дизайн
- Адаптивная вёрстка (mobile-first)
- Тёмная и светлая темы
- Анимации и микро-взаимодействия
- Глобальный поиск по всей системе
- Система уведомлений в реальном времени
- Сворачиваемый sidebar

## ТЕХНОЛОГИЧЕСКИЙ СТЕК

### Frontend:
- React 18
- TypeScript
- React Router
- React Hook Form + Zod (валидация)
- Tailwind CSS
- Vite

### Backend:
- Node.js
- Express
- REST API

### Архитектура:
- Monorepo (npm workspaces)
- Компонентный подход
- Context API для управления состоянием
- Полная типизация TypeScript
